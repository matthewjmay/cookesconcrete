Agency Jekyll theme
====================

Agency theme based on [Agency bootstrap theme](https://startbootstrap.com/template-overviews/agency/)

# How to use

### Portfolio

Portfolio projects are in `/_posts`.

Images are in `/img/portfolio`.

### About

Images are in `/img/about/`.
